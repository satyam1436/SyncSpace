# SyncSpace

A real-time collaborative whiteboard and code editor built using the MERN Stack.

## Features

- Real-time Collaborative Whiteboard
- Real-time Code Editor
- Authentication
- Room Management
- Live Chat
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- React Router
- Socket.io Client

### Backend
- Node.js
- Express.js
- MongoDB
- Socket.io
- JWT

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/satyam1436/SyncSpace.git
```

### Install Frontend

```bash
cd frontend
npm install
npm run dev
```

### Install Backend

```bash
cd backend
npm install
npm run dev
```
